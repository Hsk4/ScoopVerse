# ScoopVerse Ice Cream Website Setup Guide

## Quick Start Steps:

### Step 1: Copy Image Assets
Copy all images from `f:\Projects\Datascience&python\Django\images\images\` 
to `f:\Projects\Datascience&python\Django\Scoop-project\static\scoop_app\images\`

### Step 2: Run the Development Server
Since we can't run commands from here, open PowerShell and run:
```powershell
cd f:\Projects\Datascience&python\Django\Scoop-project
..\blog-app\.venv\Scripts\python.exe manage.py runserver
```

### Step 3: Visit the Website
Open your browser and go to http://127.0.0.1:8000/
